Creador de Elementos de Lista Dinámica

Descripción: Crea una aplicación que permita al usuario agregar elementos a una lista desordenada en el DOM.
Requerimientos:
Usa un campo de entrada y un botón para que el usuario pueda escribir el texto del nuevo elemento.
Al presionar el botón, el texto se debe agregar como un nuevo elemento <li> en una lista desordenada.
Agrega un botón de "Eliminar" junto a cada elemento de la lista, que permita eliminarlo.
Pistas:
Usa document.createElement() para crear los elementos dinámicamente.
Usa appendChild() para agregar los elementos al DOM.
